# AskSheets AI

AskSheets AI is a CSV analysis web app where users create workspaces, upload CSV files, preview data, and ask natural-language questions that return tables, summaries, and charts.

## Features

- Create isolated workspaces
- Upload CSV files
- Preview uploaded data
- Ask natural-language analysis questions
- Generate grouped sales tables
- Render simple bar charts
- FastAPI backend + Next.js frontend

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

Backend:
- FastAPI
- Pandas
- Uvicorn

## Demo Flow

1. Create a workspace
2. Upload a CSV
3. Preview rows and columns
4. Ask: "What is total sales by region?"
5. View table and chart response
